# GATHA Backend Service Utilities

## Helper Scripts

This directory contains utility scripts for managing the backend service.

### Available Scripts

1. **Create Admin User** - Bootstrap the system with an admin user
2. **Database Cleanup** - Remove old logs and notifications
3. **Seed Data** - Populate database with sample data for testing

---

## Usage

Run scripts from the backend directory:

```bash
node services/createAdmin.js
node services/cleanupDatabase.js
node services/seedData.js
```
